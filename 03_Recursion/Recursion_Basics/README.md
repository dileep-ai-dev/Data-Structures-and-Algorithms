## Key Learnings

- Every recursive function must have a base case.
- Every recursive call should move closer to the base case.
- A recursive function pauses while making recursive calls and resumes execution after the recursive call returns.
- Functional recursion builds the answer while returning.
- Parameterized recursion carries the answer through function parameters.
