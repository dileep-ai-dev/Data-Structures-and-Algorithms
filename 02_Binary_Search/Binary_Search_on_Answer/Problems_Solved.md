# Problems Solved

## LeetCode 875 - Koko Eating Bananas

- Status: ✅ Solved
- Difficulty: Medium

### Approach

- Binary Search on Answer
- Helper Function (is_possible)

### Time Complexity

O(n log m)

### Space Complexity

O(1)

### Learning

- Binary Search can be applied to the answer space.
- The helper function validates whether a candidate answer is feasible.
- If the answer is feasible, search for a smaller valid answer.
